可选大模型接口

默认模式仍然是纯静态：直接打开 index.html 就能学习 15 天课程。这里留给会部署云函数或会写代码的人，把当前课程、词表、英文占比和收藏错词发给自己的后端，让后端调用任意大模型重写故事。

当前是纯静态模式。配置接口后，才会向你的云端发送请求。

接口契约

前端只发送学习上下文，不保存也不暴露任何大模型密钥。请把 API key 放在你自己的云函数或服务器环境变量里。

POST /storyvocab/generate
Content-Type: application/json

request: {
  "day": 1,
  "theme": "campus",
  "englishDensity": 30,
  "words": [{ "word": "listen", "display": "listen to", "zh": "听" }],
  "favorites": ["listen to"],
  "reviewWords": ["depend on"]
}

response: {
  "storyText": "一篇新的故事...",
  "notes": "可选说明"
}
接口返回的故事会显示在这里，不会覆盖本地固定课程。

代码接口

开发者可以在自己的脚本里读取 window.StoryVocabAPI。它提供当前状态、当前课程词表、请求载荷构造、外部故事渲染和美音朗读入口。

const payload = window.StoryVocabAPI.buildConnectorPayload();
window.StoryVocabAPI.renderExternalStory("My generated story");
window.StoryVocabAPI.speak("listen to");

项目定位

StoryVocab 是一个可公开部署到 GitHub Pages 的故事化英语词汇学习工具。核心目标是把词放进有画面、有冲突、有情绪的文章里，而不是把词按字母表机械排列。

  • 15 天，每天 200 个混排新词。
  • 浏览器本地保存收藏、错词、熟悉度和复习日程。
  • 使用 Web Speech API 播放 en-US 美音，不内置音频文件。
  • 支持自定义词本导入、导出和主题化阅读。

隐私与安全

默认模式不需要登录、服务器、数据库或追踪脚本。学习进度保存在浏览器本地；如果接入自己的云端接口，请把 API key 放在服务器环境变量里，不要写进这个静态网页。

  • 不要在公开 issue 里粘贴 API key、密码或私有端点。
  • 安全问题请发邮件给作者，而不是公开贴细节。
  • 第三方模型服务的费用、隐私和条款由接入者自己确认。
