项目定位

StoryVocab 3000 是一个可公开部署到 GitHub Pages 的英语高频词学习工具。核心目标是把词放进有画面、有冲突、有情绪的文章里，而不是把词按字母表机械排列。

  • 15 天，每天 200 个混排新词。
  • 浏览器本地保存收藏、错词、熟悉度和复习日程。
  • 使用 Web Speech API 播放 en-US 美音，不内置音频文件。
  • 支持自定义词本导入、导出和主题化阅读。
