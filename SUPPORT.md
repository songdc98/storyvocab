# Support

## Get Help

Use GitHub issues for reproducible bugs, feature requests, documentation fixes, and learner-experience suggestions:

https://github.com/songdc98/storyvocab/issues

For collaboration, research/product discussion, educational use, internship/job opportunities, referrals, or professional introductions, copy this email address:

`dsong25@gmu.edu`

## Help The Project

If StoryVocab helps you learn or teach English vocabulary, please:

- star the repository: https://github.com/songdc98/storyvocab
- share the live demo with learners or teachers who may benefit;
- open thoughtful issues with concrete examples;
- contribute better stories, word lists, accessibility fixes, or connector examples;
- introduce the author to teams working on language learning, education tools, or human-centered AI.

## What Not To Send

Do not send API keys, passwords, private user data, student records, confidential manuscripts, or proprietary word lists unless there is a clear private agreement and a secure channel.
